# M.A.P.A. Interface (Monitorização de Apoio e Prevenção Ativa)

## Overview
A mental health screening interface focused on privacy, zero-distraction, and accessibility. 
Built using React, Vite, and Tailwind CSS v4.

## Core Features
1. **Welcome Screen**: Clean presentation with a simple start button.
2. **Questionnaire Flow**: One question per screen, big Likert scale cards.
3. **Completion Screen**: Thank you message and a nature/relaxation audio pill.

## Design Identity
- **Topology**: Vertical, centered, oversized cards.
- **Palette**: Soft earthy pastels (Sage, Peach, Cream). NO PURPLE.
- **Animations**: Subtle, slow-easing GPU-accelerated fades and slides.
